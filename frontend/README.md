# Udyara AI Frontend

User interface for interacting with the NitiAI policy agent.

---

## ⚙️ Setup

```bash
npm install
npm run dev
```

- Frontend
Runs at: http://localhost:3000

- Ensure backend is running at:
Backend: http://localhost:8000
- Frontend sends queries to /ask endpoint.

---
## Features

1. Chat-based UI
2. Policy Q&A
3. Clean, accessible design
